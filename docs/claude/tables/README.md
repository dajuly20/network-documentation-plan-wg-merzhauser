# Tabellen-Übersichten

In diesem Ordner liegen tabellarische Übersichten als Markdown (`.md`), z. B.:

- Netzwerke & VLANs (`networks.md`)
- Dienste & Systeme (`services.md`)
- Hosts & Hardware (`hosts.md`)

Die Tabellen können direkt in andere Dokumente kopiert oder per Include verarbeitet werden.
